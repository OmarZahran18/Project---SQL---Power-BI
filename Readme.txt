# Supermarket Sales Analysis

## Overview
This project analyzes supermarket sales data using SQL and presents insights through an interactive Power BI dashboard.

## Key Insights
- Total sales by branch, city, and customer type.
- Peak sales times and most popular payment methods.
- Gender-based spending patterns.
- Product lines with the highest sales and ratings.

## Tools Used
- **SQL**: Data extraction and analysis.
- **Power BI**: Data visualization and dashboard creation.

## How to Use
- Open the SQL file to review the queries.
- Use Power BI Desktop to explore the dashboard (`supermarket_sales_dashboard.pbix`).
